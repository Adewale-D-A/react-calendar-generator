# React Calendar Data Generator

> This calendar generator app is a template application mainly created for its codebase to show how a custom caledar data can be generate, displayed and manipulated to feature any element the developer choose. This project need came from a Proptech solution that had a feature where apartments' calendar data is shown displaying when maintenance are schdedule, what rooms are available/unavailable and other data that can be infered on a calendar view. This is a developer first approach to extract the code and logic for their use

### <b>Prerequisite</b>:

> TailwindCSS(CSS styling), NextJs (View Renderer). Therefore, technical knowledge of these tools are required to collaborate on this project in TypeScript.
